# Specimen Sandbox — Icon Assets

Identity: **Seed Spiral · Anchored** variant from the Specimen Sandbox
brand design pass (Claude Design handoff, 2026-05-14).

## Master assets

| File | Size | Use |
|---|---|---|
| `icon-1024.png` | 1024×1024 8-bit sRGB (no alpha) | iOS App Store hero. Ink stock + Fibonacci spiral foil seeds + iridescent pearl center. `expo.icon` in `app.json`. |
| `adaptive-foreground.png` | 1024×1024 8-bit sRGB w/ alpha | Android adaptive icon foreground. Spiral + pearl on transparent. `expo.android.adaptiveIcon.foregroundImage`. |
| `adaptive-background.png` | 1024×1024 8-bit sRGB | Android adaptive icon background. Ink stock with subtle grain. `expo.android.adaptiveIcon.backgroundImage`. |
| `monochrome.png` | 1024×1024 8-bit sRGB w/ alpha | Wear OS / notification / status bar. White spiral on transparent (system applies its own tint). `expo.android.adaptiveIcon.monochromeImage`. |
| `icon-mark.svg` | 1024 viewBox | Vector source of truth. Regenerated by `scripts/render-icons.mjs`. |

## Palette

- Ink stock: `#0a0a0c`
- Ink grain: `#1a1916` at 60% opacity
- Cream highlight: `#f1ecde`
- Cream base: `#e9e3d2`
- Foil gradient (balanced): blue → lavender → pink → cream → mint → blue
- Pearl gradient: warm cream → peach → rose → blue

## Reproduction

```bash
node scripts/render-icons.mjs
```

The `PARAMS` object at the top of `scripts/render-icons.mjs` mirrors the
design tool's tweak panel — pearl size, foil intensity, density, stock,
pearl style, halo. Edit and re-run to iterate.
