### Task 3.3: Offline state hook

**Step 1: Write the hook**

```ts
// hooks/useOnline.ts
import NetInfo from '@react-native-community/netinfo';
import { useEffect, useState } from 'react';

/** Returns `true` when the device has internet, `false` when offline. Defaults to true. */
export function useOnline(): boolean {
  const [online, setOnline] = useState(true);
  useEffect(() => {
    const unsub = NetInfo.addEventListener((state) => {
      setOnline(state.isConnected !== false && state.isInternetReachable !== false);
    });
    return () => unsub();
  }, []);
  return online;
}
```

Diagnostic checklist:
1. Is it a libdav1d / expo-image conflict? App A's `8e8d9e2` disabled libdav1d to unblock — check if app B's expo-image has the same plugin config issue.
